# Dark/Light Mode Toggle

A simple web page to switch between **dark** and **light** themes. The page detects your system theme on first visit and remembers your choice using **localStorage**.

## Features

- Automatic system theme detection  
- Toggle button to switch themes manually  
- Saves user preference across page reloads  
- Smooth transition between themes

## Usage

1. Open `index.html` in your browser.  
2. Click the **Toggle theme** button to switch modes.  
3. Your preference is saved automatically.

## Files

- `index.html` – main page  
- `style.css` – styles with CSS variables and transitions  
- `script.js` – theme logic using JavaScript and localStorage
